This is a Better Motherfucking Website

It’s readable. It’s fast. It doesn’t waste your time with bloated nonsense. There’s no JavaScript, no cookie banner, no tracking, no fonts from Google. Just HTML and a bit of CSS.

Why?

Because you don’t need a framework to write a paragraph of text. Just write the damn thing. Use semantic HTML and make it look okay with some minimal CSS.

How?

Type it in a text editor. Save it as .html. Upload it. Done. If you want to make a better web, start with less bullshit.
